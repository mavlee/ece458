Keystroke Dynamics Test

Keystroke dynamics is a form of biometrics - using each person's body and habits as an authentication system or an extra layer of security. In this case, we use how a person types to determine if they are indeed the person who the password is meant for.

To try: enter in some text (press enter to enter it in again). We recommend entering in the text at least 10 times. After that, try typing the same text in the tester box, and see if you are verified. Allow someone else to try, and see if they are verified. Don't worry about making mistakes when typing, these will be taken into account.
